Política de Privacidade - RDC Home

Data de Vigência: 30 de outubro de 2025

O RDC Home é um app de automação residencial que opera localmente em sua rede Wi-Fi. Não coletamos dados pessoais, não enviamos informações para servidores externos e não usamos trackers de terceiros. Todos os dados (como status de relés e sensores) ficam no seu dispositivo ou rede local.

Informações Coletadas

O app não coleta dados pessoais (nome, email, localização, etc.). Ele apenas lê dados locais dos ESPs (ex.: temperatura, umidade, status de relés) via MQTT na sua rede Wi-Fi. Esses dados não saem do seu dispositivo ou rede local.

Uso de Dados

Os dados dos sensores são usados apenas para exibir no app (ex.: mostrar temperatura na tela). Não há análise, compartilhamento ou armazenamento em nuvem.

Compartilhamento de Dados

Não compartilhamos dados com terceiros. O app não usa serviços de publicidade ou analytics.

Segurança

Usamos MQTT local (porta 1883) com autenticação opcional. Recomendamos senha no Mosquitto para segurança. Não há criptografia externa, pois é local.

Direitos dos Usuários

Você pode acessar/deletar dados a qualquer momento (ex.: limpar cache do app). Para suporte, contate [seu email].

Mudanças na Política

Atualizaremos esta política se necessário e notificaremos via app.
